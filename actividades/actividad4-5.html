YouTube video ID
YouTube mostrará un ID (como EF81Pb5IWoU), al guardar (o reproducir) yn vídeo. Puede utilizar este ID y hacer referencia a su vídeo en el código HTML.

Reproducción de un vídeo de YouTube en HTML:
Para reproducir el vídeo en una página web, haga lo siguiente:

  • Sube el vídeo a YouTube
  • Anote el ID de video
  • Defina un elemento <iframe> en su página web
  • Deje que el atributo src apunte a la URL del vídeo
  • Utilice los atributos width y height para especificar la dimensión del reproductor
