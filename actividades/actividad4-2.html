La etiqueta video: Paraa especificar la ubicación del archivo que se reproducirá, puede usar el elemento <source> dentro del elemento <video>. (Esto debe reemplazar el atributo src en la etiqueta <video> de apertura.) También puede utilizar varios elementos <source> para especificar que el video esta disponible en diferentes formatos. (Debido a un error en el iPad, debe proporcionar el video MP4 como el primer formato. De lo contrario, puede que no inicie.)

src: este atriburo especifica la ruta de acceso al video

b>type: se debe utilizar este atriburo para decirle al navegador que formato es el video. De lo contrario descargará parte del video para ver si puede reproducir el archivo. (que tomará tiempo y ancho de banda).

codecs: el códeoc que se usó para codificar el video se suministra dentro del atributo type. Tenga en cuenta el uso de comillas simples, así como comillas dobles en el atributo de tipo, cuando se suministra.
